¿Qué es Angular?

  • Framework para para aplicaciones web desarrollado en.
  • Código abierto.
  • Mantenido por Google.
  • Objetivo: aumentar las aplicaciones basadas en navegador con capacidad de .
  • Se basa en clases tipo "Componentes", cuyas propiedades son las usadas para hacer el binding de los datos. En dichas clases tenemos propiedades (variables) y métodos (funciones a llamar).

Instalación

Para instalar Angular en tu sistema local necesitas:

  • Node.js
  • npm package manager

Angular CLI

Utilizado para crear proyectos, generar aplicaciones y librería de código, además de tareas de desarrollo como pruebas o implementación.

  • Para instalar CLI de Angular, abre una terminal y ejecuta el siguiente comando: npm install -g @angular/cli

Workspace y aplicacion inicial

  1. Ejecutar el comando CLI ng new y asignarle un nombre deseado, ejemplo: my-app
  2. El comando ng new solicita información sobre funciones que serán incluidas. Se debe aceptar los valores predeterminados presionando la tecla Enter o Volver.

Ejecutar la aplicación

Angular CLI ya trae incorporado un servidor para que el usuario pueda construir su aplicación de manera local.

  1. Debemos navegar a la carpeta donde se encuentra el workspace creado anteriormente my-app
  2. Ejecutar el comando cd my-app para ubicarnos dentro de nuestro proyecto Angular y luego ejecutamos el comando ng serve --open

ng serve inicia el servidor, y modifica la aplicación a medida que se realizan cambios. La dirección de este servidor es http://localhost:4200/

Conceptos clave para entender Angular

  • Components
  • Templates y directives
  • Dependency injection

¿Cuándo usar Angular?

Cuando se tienen aplicaciones web con servicio online, formularios y procesos con respuesta inmediata del servidor.

También para el desarrollo de apps tanto híbridas como nativas, ya que son varios los framework para construir aplicaciones móviles que usan Angular, entre ellos: NativeScript o Ionic.

Volver

TypeScript

TypeScript es un lenguaje de programación libre y de código abierto desarrollado y mantenido por Microsoft. Es un superconjunto de JavaScript, que esencialmente añade tipos estáticos y objetos basados en clases. TypeScript es usado para desarrollar aplicaciones JavaScript que se ejecutarán en el lado del cliente o del servidor, o extensiones para programas (Node.js y Deno).

MVC

El Modelo Vista Controlador es un estilo de arquitectura de software que separa los datos de una aplicación, la interfaz de usuario, y la lógica de control en componentes distintos.
