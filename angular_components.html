Components

Son el bloque principal para construir aplicaciones en Angular.

Un componente incluye una clase TypeScript con un @Component() decorator, una plantilla HTML y estilos.

El @Component() va a especificar la siguiente información:

  • Un selector CSS que define como se usa el componente en la plantilla. Elementos que coinciden con este selector, pasan a ser instancias del componente
  • Una plantilla HTML que le dice a Angular como debe renderizar el componente
  • Un set opcional de estilo CSS que define la apariencia de los elementos de la plantilla HTML

Creación

La mejor manera es usando el Angular CLI, pero se puede hacer de manera manual.

  1. Desde una terminal debemos ir a la ruta donde se encuentra nuestra aplicación
  2. Ejecutamos el comando ng generate component <nombre-componente>

Este comando creará:

  • Un directorio llamado igual que el componente
  • Un archivo del componente, <nombre-componente>component.ts
  • Una plantilla, <nombre-componente>.component.html
  • Un archivo CSS, <nombre-componente>.component.css
  • Un archivo de especificación de prueba, <nombre-componente>.component.spec.ts
